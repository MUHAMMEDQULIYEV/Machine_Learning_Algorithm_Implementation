# Custom Linear Regression from Scratch

A clean, object-oriented implementation of Linear Regression using raw NumPy and Gradient Descent. 

## Features
- **Vectorized Gradient Descent**: Optimized matrix operations via NumPy dot products.
- **Persistence**: Save and restore model states using binary serialization (`pickle`).
- **Clean API**: Matches standard machine learning framework syntax (`.fit()`, `.predict()`).
